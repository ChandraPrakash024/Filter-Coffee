### **IPv4 Packet Format**

```plaintext
+--------+--------+-----------------------+-------------------+
| Version (4) | IHL (4) | Type of Service (8)               | Total Length (16) |
+--------+--------+-----------------------+-------------------+
| Identification (16)                | Flags (3) | Fragment Offset (13)          |
+------------------------------------+-----------------------+
| TTL (8)        | Protocol (8)       | Header Checksum (16)                     |
+-----------------+-------------------+-----------------------------------------+
| Source IP Address (32 bits)                                              |
+---------------------------------------------------------------------------+
| Destination IP Address (32 bits)                                         |
+---------------------------------------------------------------------------+
| Options (Variable)                                    | Padding (Optional) |
+---------------------------------------------------------------------------+
| Data (Variable)                                                         |
+---------------------------------------------------------------------------+
```

### **Layer Summary: ISO/OSI Model**

```plaintext
+-------------------------------------------------------------------------+
| Layer 7: Application (e.g., HTTP, FTP, SMTP, DNS)                       |

+-------------------------------------------------------------------------+
| Layer 5: Session (e.g., NetBIOS, PPTP)                                  |
+-------------------------------------------------------------------------+
| Layer 4: Transport (e.g., TCP, UDP)                                     |
+-------------------------------------------------------------------------+

+-------------------------------------------------------------------------+
| Layer 2: Data Link (e.g., Ethernet, PPP)                                |

+-------------------------------------------------------------------------+
```
